=head1 NAME

wg-meta - Manage Wireguard using Perl

=head1 VERSION

#VERSION#

=head1 DESCRIPTION

An approach to add metadata to the main wireguard config. Runs on plain (core-)Perl, no external storage dependencies
and integrates seamlessly in your existing setup.

=over 12

=item *

A powerful parser/writer for Wirguard configuration files L<Wireguard::WGmeta::Wrapper::Config> with routines to add,
remove, disable/enable and modify Wireguard interfaces and peers -  also usable as standalone.

=item *

A parser for C<wg show> output L<Wireguard::WGmeta::Wrapper::Show>, also usable as standalone.

=item *

A Cli interface L<Wireguard::WGmeta::Cli>

=back
